My Hobby

Hobi Saya adalah bermain game, sepak bola, basket, dan renang. Alasan saya memilih untuk menjadi game developer karena saya sangat suka sekali bermain game, dan saya terinspirasi dari permainan yang saya mainkan. Sehingga saya sangat sekali ingin belajar dan membuat game sendiri. Dan membuat game itu tidaklah mudah, butuh beberapa tahun untuk memahami konsep dari game engine yang digunakan. Saya sendiri belajar membuat game selama kurang lebih 3 Tahun, hasilnya pun masih kurang begitu bagus. Akan tetapi Saya tidak akan menyerah sampai disini saja, saya akan terus semangat untuk menggapai cita-cita saya sampai tua nanti.
